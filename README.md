# 🎓 Udemy Courses Data Analysis

## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Udemy Courses Dataset** using Python. The analysis explores course pricing, subscriber trends, subject popularity, course levels, reviews, and other key metrics to uncover meaningful insights.

The project demonstrates the complete data analysis workflow, including data cleaning, preprocessing, visualization, and interpretation.

---

## 📂 Dataset

**Dataset Name:** Udemy Courses Dataset

**Dataset File:**

```
5-udemy_courses.csv
```

The dataset contains information such as:

- Course Title
- Subject
- Price
- Number of Subscribers
- Number of Reviews
- Number of Published Lectures
- Level
- Published Date
- Content Duration
- Free/Paid Status

## 🎯 Objectives

- Clean the dataset
- Handle missing values

- Analyze pricing trends
